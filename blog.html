DOM -QUEST
What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web

How do you select an element from the DOM?

The querySelector method allows you to select the first element that matches a specified CSS selector(s) in the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to manage events for multiple elements more efficiently by taking advantage of event propagation

How do you manipulate an element's attributes and styles using the DOM?

getAttribute() and setAttribute() getAttribute(attributeName): Retrieves the value of the specified attribute. setAttribute(attributeName, value): Sets or updates the value of the specified attribute.
